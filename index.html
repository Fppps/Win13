Windows 13 Restore Realized

Version 1.0.0

Welcome to the first major feature update for the Windows 13 Concept. This release focuses on fundamental system improvements and new core applications:

  • **Window Resizing:** Windows are now resizable, offering greater flexibility in workspace management.
  • **Interactive File System:** A fully functional File Explorer allows for intuitive navigation and basic file operations (create files).
  • **Text Editor (Notepad):** Introduction of a versatile text editor supporting both `.txt` and `.py` file types.
  • **CPU Monitor & Simulator:** A new application to monitor virtual CPU registers and memory, with the ability to load and execute simple assembly-like programs.
  • **Enhanced Dock & Start Menu:** Improved accessibility for key applications directly from the dock and an updated Start Menu.
  • **Theming Options:** Easily switch between light and dark desktop themes.
  • **Basic Terminal Functionality:** Expanded commands for system interaction.

Summary: This update lays the groundwork for a more interactive and feature-rich operating system concept, bringing essential tools and improved user experience.

Windows + Linux Functionality & Supported Files

Version 1.0.5

This update further enhances the hybrid OS experience by integrating more command-line tools and improving file handling, offering a powerful blend of Windows and Linux features:

  • **Expanded Terminal Commands:** Added common Linux commands like `pwd` (print working directory), `echo`, `whoami`, and `date` to the terminal for greater versatility.
  • **Improved File System Navigation:** The `cd` command in the terminal now supports absolute paths (e.g., `/C:/Users/You/Documents`) and more robust relative path handling.
  • **Enhanced File Operations:** Terminal `rm` command now includes a basic check for non-empty directories (though it only supports removing empty ones for simplicity).
  • **Unified Text Editor:** The "Notepad" app now intelligently adapts its title and icon based on the file extension (.txt for Notepad, .py for Code Editor), providing a more cohesive experience. The editor styling also includes a subtle "Linux" hint through its dark background and monospace font, reminiscent of popular Linux text editors.
  • **Context Menu Integration:** Right-clicking on the desktop now brings up a context menu with quick links to personalization settings, mirroring a classic desktop experience.
  • **Interactive Clock:** The desktop clock is now clickable, directly opening the "About" section in the Settings application.

Summary: Version 1.0.5 deepens the hybrid OS feel by bringing more Linux-like command-line power and enhancing core application usability, making the system more flexible and powerful.

Updated Login and Account Management

Version 1.0.6

This update significantly enhances user account security and management:

  • **Password Encryption:** User passwords are now securely hashed using SHA-256 before being stored, providing improved data protection.
  • **Multi-User Support:** The system now supports multiple user accounts stored in a client-side database (`localStorage`).
  • **Persistent Usernames:** Upon logout, the user's password is "shut down" (cleared) from storage for security, but their username remains, allowing for easier re-login or account recovery (recovery not implemented).

Summary: Version 1.0.6 introduces enhanced login security and a more robust multi-user account system for a more personalized and secure experience.

Desktop Layout Refinement

Version 1.0.7

This update introduces a refreshed desktop layout, moving away from a traditional taskbar to a more fluid, modern dock-based system:

  • **Taskbar Removed:** The fixed full-width taskbar at the bottom of the screen has been removed for a cleaner desktop aesthetic.
  • **Floating Dock:** Essential system and pinned application icons are now housed in a visually appealing, centrally-located floating dock at the bottom of the screen.
  • **Relocated Clock & Theme Toggles:** The interactive clock (with its dynamic year display) and the theme toggles have been moved from the former taskbar to the top-right corner of the desktop, integrating them more seamlessly into the background.
  • **Improved Desktop Cleanliness:** This change aims to provide more screen real estate for open windows and a less cluttered overall appearance.

Summary: Version 1.0.7 reimagines the desktop interface, focusing on minimalist design and intuitive access to applications via a streamlined floating dock.

Customizable Date and Time Display

Version 1.0.8

This update refines the desktop clock display, offering more granular control over its appearance:

  • **Separated Time and Date:** The time and date components of the desktop clock are now rendered in separate custom `div` elements within the clock container. This provides individual styling flexibility for each part.
  • **Enhanced Styling Control:** Developers and users can now apply distinct CSS classes and styles to the time and date elements independently, allowing for more tailored visual presentation.
  • **Maintained Functionality:** All existing interactivity, such as clicking the clock to open settings and toggling the year display, remains fully functional with the new structure.

Summary: Version 1.0.8 improves the customizability of the desktop's time and date display by separating these elements into individual, style-friendly `div` containers.

Calculator Application

Version 1.0.9

A new essential utility has been added to the Windows 13 Concept:

  • **Full-Featured Calculator:** A basic calculator application is now available, supporting addition, subtraction, multiplication, and division.
  • **Intuitive Interface:** The calculator features a clear display and a standard keypad layout for ease of use.
  • **Integrated into Dock:** The Calculator app can be quickly launched directly from the floating dock at the bottom of the desktop.
  • **Window Management:** Like other applications, the Calculator window supports dragging and resizing.

Summary: Version 1.0.9 introduces the Calculator app, enhancing the system's utility and providing a convenient tool for everyday calculations.

Task Manager / Process Viewer

Version 1.1.0

A new "middle-end" application has been integrated to provide centralized control over running processes:

  • **Process Monitoring:** The Task Manager lists all currently open application windows, displaying their names and unique IDs.
  • **End Task Functionality:** Users can select any running application and force it to close using the "End Task" button.
  • **Real-time Updates:** The process list can be refreshed to show the most current active windows.
  • **Enhanced System Control:** This feature offers a deeper level of interaction with the simulated operating system, allowing for better resource management and troubleshooting.

Summary: Version 1.1.0 introduces the Task Manager, significantly improving system control and providing a modern, centralized interface for managing running applications.

Sticky Notes & Updater Hub with Installers

Version 1.2.0

This update brings new desktop productivity and a system for installing applications:

  • **Sticky Notes:** A new desktop application allowing you to create, save, and display multiple sticky notes directly on your desktop. Notes persist across sessions.
  • **Updater Hub:** A dedicated application for managing software installations. It lists available applications (like Web Browser and Media Player) that can be "installed."
  • **Simulated Installers:** Clicking 'Install' in the Updater Hub simulates a software installation, making the new applications (Web Browser, Media Player) available in the Start Menu and Dock.
  • **Refined UI/UX:** Enhanced styling for windows, buttons, and dock icons for a more polished, modern aesthetic with improved transitions.

Summary: Version 1.2.0 introduces desktop sticky notes and a functional Updater Hub, bringing a new layer of interactivity and simulated software management to the system.
